### 設定 Git

git config --global user.name "YOUR NAME"
git config --global user.email "YOUR EMAIL"

## 安裝 git

```bash
    sudo apt install -y git
```

## 安裝 Just

```bash
sudo apt install -y just
```

## 安裝GitHub CLI

```bash
sudo apt install -y gh
```

## 安裝 node.js 和 pnpm

```bash
sudo apt install -y nodejs npm
npm install -g pnpm
```

# 驗證

```bash
node --version
npm --version
git --version
gh --version
just --version
```

## 環境變數

```bash
cp .env.example .env
```
